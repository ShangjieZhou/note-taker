## Before you begin
Make sure you have the following on your computer:
1. Node.js (version 16.x or later)
2. npm

## Local setup
1. Clone the repo
```bash
git clone <repository-url>
```

2. Go to the project directory
```
cd note-taker
```

3. Install dependencies
```
npm install
```

## Run the app
```
npm run dev
```
